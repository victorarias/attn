Review Loop Sketches

Three directions for making the review loop feel intentional, hidden-by-default, and readable at a glance.

These sketches assume the review loop only exists for Claude sessions, uses color as the primary state cue, and keeps reviewer output tucked away until the user asks for it or input is required.
