## Implementation Notes

All measurements are in pixels unless otherwise specified.

Colors reference the CSS custom properties defined in visual-design.md:
- `var(--bg-primary)` = #0a0a0b
- `var(--text-primary)` = #e8e8e8
- etc.

Spacing follows 4px baseline grid:
- 4px, 8px, 12px, 16px, 24px, 40px

Border radius follows scale:
- 3px (kbd), 4px (buttons), 6px (items), 8px (cards), 12px (modals)

All transitions use ease-out for entering, ease-in for leaving, ease-in-out for toggles.
