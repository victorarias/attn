    A prototype of the attn durable markdown vault. Try these:
    • Click a file Open any note in the file tree (left). Kind dots: amber = journal, blue = memory.
    • View / Edit Toggle the center header. Edit gives raw markdown + Save / save status.
    • Select text Right-click selected text (or use the floating button) → Send to chief.
    • Tile / Fullscreen Top-bar segmented control. Tile shows the KB beside agent terminals.
    • Click a /link Root-absolute links and frontmatter sources navigate the tree.
    • Outline / Backlinks Right panel — jump to headings, see who links here.
