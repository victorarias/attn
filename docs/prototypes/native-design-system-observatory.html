Color carries identity or status, never decoration. Six ink steps from the deepest space-black to a quiet surface tone, five moonstones for type, a single sodium accent for emphasis, three stars naming the agents, and seven classifications naming what they are doing.

§ 1.1 · INK
Six surface inks — deepest to lightest
Used for backgrounds, cards, raised surfaces, and borders. Cool blue-black with a faint atmospheric tint; never pure black, never tinted toward purple. Each step is approximately 1.4× the luminance of the previous.

§ 1.5 · CLASSIFICATION
Seven states — the observation classes
Borrowed from observational vocabulary. Transit is something passing in front (active work). Occulted is something blocked from view (waiting). Halted is an emergency hold (approval). Each gets a single hue, a single dot, and a single Latin label.

Plate 05 — recto

Motion, in three registers

aurora caret
breath pulse
drift background

Motion narrates only when state changes the user did not request. Three registers: a single accent caret that breathes; a state indicator that pulses on live work; and the starfield that drifts so slowly you almost don't notice — but you do.

Aurora caret
The cursor never blinks. It pulses softly between two halo intensities, signaling "alive" without strobe.
attn
duration · 2.4s · ease-in-out · opacity 0.85 → 1.0
Breath pulse
Live state indicators (working, approval, waiting) breathe at speeds that match urgency — slower for working, faster for halted.
duration · 3.6s working / 2.4s halted · scale 0.95 → 1.05
Starfield drift
Two parallax layers translate across the page over many minutes. Almost imperceptible, but anchors the page in a slowly-moving frame.
